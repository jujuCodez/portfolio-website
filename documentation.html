Introduction / Project Overview / Objectives

This project demonstrates the development of a multi-server HTML website utilizing Python, Node.js, and React.js. Each server is configured to handle specific functionalities within the website, ensuring efficient and modular operation. The primary objective is to showcase proficiency in implementing diverse web technologies to create a fully functional, requirement-compliant site. The project also integrates features from previous modules, optimizing them for better performance and user experience.

Summary Table

Project Component Details
Batch File A batch file is provided to automatically start all three servers (Python, Node.js, and React.js) on their respective ports.
Home Page The home page includes a header, navbar, footer, and main content area. Additional information is available as the user scrolls down.
Personal Profile Page The profile page includes tooltips for enhanced user interaction.
Photo Album The photo album uses the jQuery slick carousel to display a slideshow. Users can toggle the slideshow on and off.
Portfolio This section showcases various projects. Clicking on a project opens a PDF with detailed information. A link to the job order is also provided.
Sign Up Page Utilizes Node.js to save user information in a JSON file. It includes validation to check if a username is already taken and allows users to upload a profile photo to the server.
Article 1 - Videos This article discusses personal projects and includes an embedded video, meeting the embedded media requirement.
Password-Secured Pages Articles 2 and 3 are protected by a login system. Content only becomes visible after successful authentication. The uploaded image and username are displayed, along with data visualizations (bar, pie, and line charts) for the Japanese and Filipino housing markets.
Professional Services Accessible from the portfolio page, this section includes a form for potential clients to hire services. The form, built with Node.js, supports file uploads for attaching detailed project PDFs. Once submitted, it provides a summary and stores the order in a JSON file.
Article 4 - Node.js This article demonstrates retrieving HTML content from the Node.js filesystem and displaying it within the site. Users can upload and view HTML files. It also includes a scientific calculator implemented with jQuery, modals, and draggable elements.
Article 5 - React.js This article showcases a React.js application, including interactive components such as a card that explains React.js, a component that tracks clicks and stores the highest score using the browser's localStorage API, and a flipping card demonstrating React components.
React JS App Structure The React.js section is structured using create-react-app, including functional components and event handling without third-party components. JSX is effectively used for rendering the UI.
